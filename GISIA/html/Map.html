Progressive Web App

Thank you for visiting Resource Data Inc.'s sample Progressive Web Appdivcation (PWA). Progressive Web Apps are HTML / Javascript websites that behave a bit more like what you would expect from a native app on an I-Phone or an Android device. PWA's for example typically have the following properties:
  • Findable in the App Store
  • Have an icon on your phone or tablets Home Screen
  • Fully responsive and built "mobile-first" for touch
  • Work offline or in a semi-connected state
  • Can Receive notifications
  • Can take large amounts of data offline
  • Permit background processing
  • Access hardware features and sensors (e.g. GPS / Camera)
