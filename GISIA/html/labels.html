Dynamic Multi-line Labeling with Related Data

Experiement labeling electric service points using multi-line expressions. The following fields are available: FIRSTNAME, LASTNAME,PHONE,ADDRESS,CITY,STATE,and ZIP.

You can also aggregate on related data using the RELATED tag. For each service point, there are 12 months of related electric usage data available in the "USAGE" field. Valid operations on the data are MAX,MIN,SUM,and AVG.
